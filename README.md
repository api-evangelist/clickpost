# ClickPost (clickpost)

ClickPost is a logistics and supply chain platform that aggregates 500+ carrier integrations, multi-channel customer notifications, and 50+ storefront/OMS/WMS connectors behind a unified REST API. The platform covers carrier recommendation, order creation (single and multi-piece), serviceability, manifesting, pickups, real-time tracking with webhooks, proof of delivery, NDR management, returns, and expected delivery date forecasting for both Indian domestic and international shipments.

## APIs

### ClickPost API
REST API providing carrier recommendation, order creation, cancellation, serviceability, manifesting, pickup scheduling, tracking (polling and webhooks), proof of delivery, NDR action management, returns, expected delivery date prediction, PUDO services, and rider/quick-commerce operations. Token-based authentication.
